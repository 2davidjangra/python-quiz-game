# Interactive Quiz Game 🧠 🎮

A sleek, console-based Quiz Game application built using Python. This project tests general knowledge (or specific topics) through multiple-choice or true/false questions, providing an engaging user experience directly in the terminal.

This project demonstrates core programming skills like handling data structures (dictionaries and lists), implementing loops, condition checks, and formatting terminal outputs.

---

## 🚀 Features
- **Instant Feedback:** Displays whether the user's answer is correct or incorrect immediately after submission.
- **Live Score Tracking:** Dynamically updates and keeps track of the player's score throughout the game.
- **Final Performance Summary:** Provides a detailed breakdown of total correct answers and a final percentage score at the end.
- **Case-Insensitive Inputs:** Gracefully handles user inputs (e.g., accepting 'A' or 'a' automatically) to prevent accidental errors.

---

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** None (Built entirely on native Python data structures)
